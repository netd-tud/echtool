ECH TOOL 
===

ECH handshakes without ECH configuration in HTTPS RRs and fine-grained control over ECH handshakes. 


```
Usage:
  echtool [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  conn        Connect to a server offering a provided ECH config
  dech        Decode an ECH configuration list
  greasy      Probe a server's ECH support via a GREASE handshake
  help        Help about any command
  test        Longitudinal ECH experiments (grace period, config variations)

Flags:
  -h, --help      help for echtool
  -v, --version   version for echtool

Use "echtool [command] --help" for more information about a command.
```
